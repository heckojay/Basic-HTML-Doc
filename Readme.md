# HTML User Documentation

## Overview

This guide is made for beginners to create and style a basic website in HTML and CSS. This guide walks through each individual step with images and gifs while providing tips and warning that will help beginners.

## What The Users Will Learn

- Download and set up VS Code
- Create a basic webpage using essential HTML tags
- Style basic webpage using CSS
- Preview your webpage live in a browser as you write it using VS Code's Live Preview extension

## Technologies Used

- VS Code
- HTML
- CSS
- MKDocs Material
- Markdown

## Purpose

The purpose of this guide is to provide clear information while explaining unfamiliar terms to beginners, building a basic website, and helping them fully understand their actions.
