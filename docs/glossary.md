## Glossary

| Term | Description |
|------|-------------|
| `Web Browser` | :material-web: Software used to view websites (e.g., Google Chrome). |
| `Extension` | :material-puzzle: An add-on that adds new features to Visual Studio Code. |
| `Emmet` | :material-lightning-bolt: A VS Code feature that expands shortcuts like `!` into full HTML templates. |
| `File Path` | :material-folder: The location of a file on your computer (e.g., `images/stp1.png`). |
| `Live Preview` | :material-monitor-eye: A tool that shows your webpage updating in real time as you edit code. |
| `GitHub` | :material-github: A platform used to store and share your code online. |
