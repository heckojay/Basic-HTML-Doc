| Term | Description |
|------|-------------|
| `Web Browser` | Software used to view websites (e.g., Google Chrome). |
| `Extension` | An add-on that adds new features to Visual Studio Code. |
| `Emmet` | A VS Code feature that expands shortcuts like `!` into full HTML templates. |
| `File Path` | The location of a file on your computer (e.g., `images/stp1.png`). |
| `Live Server` | A tool that shows your webpage updating in real time as you edit code. |
| `GitHub` | A platform used to store and share your code online. |
