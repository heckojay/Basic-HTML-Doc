| Issue              | Reason of Issue                     | Solution                                                                 |
|-------------------|-------------------------------------|--------------------------------------------------------------------------|
| Images not showing | Broken link or incorrect file path | Ensure the URL or [file path](glossary.md#file-path) is correct.      |
| Live Server not working | Live Server extension not installed or file not saved | Live Server extension not installed and ensure that live server is installed.
| Changes not appearing | File not saved | **Press** [Ctrl^S]  (Press Both Ctrl and S together).
